2 시간복잡도를 나타내는 방법 => 빅오, 오메가 , 세타
	빅오 : 최악시간
	오메가 : 최상시간
	세타 : 평균시간
	
	
	퀵정렬 : 분할 정복을 이용하여 정렬을 수행하는 알고리즘.
	
		특징
		1. 랜덤배열에서 빠른 정렬 속도를 보인다.
		2. 피벗을 선정하는 방법에 따라 속다가 달라진다.
		3. 순열이나 역순의 경우 매우 느린 속도를 보인다. (정렬이 되어있을 경우) 왜냐! =>  분할이 안되거든 
		4. 재귀함수 기반으로 구현시 복잡하게 생각될 수 있다.
		
		* pivot point : 기준이 되는 값 하나
		
		분할 횟수 * 각 요소의 갯수만큼 비교  
		요소가 31개라고 가정할 경우, 분할횟수 = 15,7,3,1=>4번
		즉, ex) 31개의 원소라고 가정할경우 4 = log2(31)    
		분할 횟수 => k = log2n
		각 요소의 갯수 = n
		n * log2n = nlog2n
